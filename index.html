Duke - A Chrome Extension project

What the hell is Duke?

Small addon to debug Giosg chat

What can it do?

  • Detect Giosg script on page
  • Detect known compatibility issues
  • Detect Rule engine
  • Detect Rooms
  • Show a list of rules and show which rules are active
  • Show current shopping cart content
